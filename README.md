# 🎓 西电评教一键完成脚本（Tampermonkey）

> 适用于西安电子科技大学 [ehall 评教系统]  
> ✅ 自动勾选 **"非常满意"**  
> ✅ 自动填写评语 **"老师辛苦了"**  
> ✅ 兼容新版伪单选框（基于 `data-x-dafxsm` 精准识别）  
> ✅ 无需手动操作，1 秒完成整页评教 🚀

## 🛠 使用前提

- 浏览器：Chrome / Edge / Firefox  
- 已安装 [Tampermonkey 扩展]油猴  
- 已登录西电统一身份认证（进入评教页面）
- 注意：如遇单选框无法选中，请阅读源码中注释

---

## 🔧 使用步骤

1. 点击浏览器Tampermonkey 图标 → “添加新脚本...”  
   粘贴 [脚本源码](./code.user.js)至脚本中
   并ctrl+s 保存脚本
2. 打开评教页面进入页面即可自动评教  
   如果需要多次评教回到主页重新进入评教页面即可
